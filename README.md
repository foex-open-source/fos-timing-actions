<h4>Overview</h4>
<p>The <strong>FOS - Timing Actions</strong> dynamic action plug-in is our solution for controling the timing and frequency of the actions contained within a Dynamic Action such as debounce, delay or throttle. It also includes a Timer to automate the repetition of your dynamic actions.
</p>
